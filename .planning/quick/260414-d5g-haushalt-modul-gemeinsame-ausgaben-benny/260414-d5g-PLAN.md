  zeitraum_von TEXT,
  zeitraum_bis TEXT,
  abrechnung_id INTEGER REFERENCES haushalt_abrechnungen(id) ON DELETE SET NULL,
  created_at TEXT NOT NULL DEFAULT (datetime('now')),

**2. haushalt.routes.ts erstellen** (Pattern wie contracts.routes.ts):

- `import { Router } from 'express';` + `import db from '../db/connection';`
- `const router = Router();`

Endpunkte:

**GET /** — Eintraege laden
- Query-Param `abrechnung_id`: wenn `'null'` oder fehlend → `WHERE abrechnung_id IS NULL`, wenn Zahl → `WHERE abrechnung_id = ?`
- Sortierung: `ORDER BY datum DESC, created_at DESC`
- Alle Felder zurueckgeben

**POST /** — Eintrag erstellen
- Body validieren: datum, betrag, beschreibung, kategorie, bezahlt_von sind Pflicht
- eintrag_typ default 'ausgabe', aufteilung_prozent default 50
- zahlungsart, zeitraum_von, zeitraum_bis optional (null wenn nicht gesetzt)
- Return: erstellter Eintrag mit 201

**PUT /:id** — Eintrag bearbeiten
- Existenz pruefen, 404 wenn nicht gefunden
- Nur uebergebene Felder updaten (wie contracts.routes.ts Pattern: body.feld !== undefined ? body.feld : existing.feld)
- updated_at = datetime('now')
- Return: aktualisierter Eintrag

**DELETE /:id** — Eintrag loeschen
- Existenz pruefen, 404 wenn nicht gefunden
- Nur Eintraege ohne abrechnung_id loeschen (offene). Wenn abrechnung_id gesetzt → 400 "Abgerechneter Eintrag kann nicht geloescht werden"
- Return: 204

**GET /saldo** — Saldo berechnen
- Alle offenen Eintraege laden (abrechnung_id IS NULL)
- Fuer jeden Eintrag:
  - eintrag_typ='ausgabe' + bezahlt_von='benny': julia_schuldet += betrag * (100 - aufteilung_prozent) / 100
  - eintrag_typ='ausgabe' + bezahlt_von='julia': benny_schuldet += betrag * aufteilung_prozent / 100
  - eintrag_typ='geldübergabe' + bezahlt_von='benny': benny_hat_gegeben += betrag (Julia schuldet weniger)
  - eintrag_typ='geldübergabe' + bezahlt_von='julia': julia_hat_gegeben += betrag (Benny schuldet weniger)
- Net-Saldo = (julia_schuldet - benny_schuldet) - (julia_hat_gegeben - benny_hat_gegeben)
- Positiv = Julia zahlt Benny, negativ = Benny zahlt Julia
- Return: `{ saldo: number, julia_schuldet: number, benny_schuldet: number, offene_eintraege: number }`

**GET /abrechnungen** — Abrechnungen-Liste
- `SELECT * FROM haushalt_abrechnungen ORDER BY datum DESC`
- Fuer jede Abrechnung: Anzahl Eintraege mitzaehlen via Subquery oder nachtraeglich
- Return: Array mit { ...abrechnung, eintraege_count: number }

**GET /abrechnungen/:id/eintraege** — Eintraege einer Abrechnung
- `SELECT * FROM haushalt_eintraege WHERE abrechnung_id = ? ORDER BY datum DESC`

**POST /abrechnungen** — Abrechnung erstellen
- Body: titel (Pflicht), notiz (optional)
- Saldo berechnen (gleiche Logik wie GET /saldo)
- INSERT in haushalt_abrechnungen mit datum=date('now'), ausgleich_betrag=berechneter Saldo
- UPDATE haushalt_eintraege SET abrechnung_id = {neue_id} WHERE abrechnung_id IS NULL
- Return: erstellte Abrechnung mit 201
